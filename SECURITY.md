## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do NOT open a public issue.**

Instead, email: **security@blackroad.io**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and provide a timeline for resolution.

## Security Measures

- All GitHub Actions pinned to specific commit hashes
- Dependabot enabled for automated dependency updates
- CodeQL analysis runs on every push and weekly
- Security headers enforced via Cloudflare Workers
- Content Security Policy (CSP) configured
- HTTPS enforced with HSTS preload
